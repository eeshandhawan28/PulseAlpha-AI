What gets built
Frontend (new)
  • Next.js 14 app in services/frontend/
  • TypeScript + Tailwind CSS + shadcn/ui
  • Analyze page: SSE stream consumer, step tracker, metric cards, markdown renderer
  • History page: table with search + stance filter
  • Sidebar with model accuracy footer badge
Backend additions (small)
  • New GET /analyze/stream SSE endpoint that emits progress events + report chunks
  • New GET /history endpoint returning saved runs
  • New POST /analyze saves each run to history.json
  • CORS enabled for localhost:3000
SSE Event Format
  • {"type":"step","node":"ingest","done":true}
  • {"type":"metrics","stance":"bullish","confidence":0.82}
  • {"type":"chunk","text":"Reliance shows…"}
  • {"type":"done","run_id":"abc123"}
Not included
  • Authentication — local use only
  • Backtest UI — CLI only (python -m worker.backtest)
  • Charts / candlestick graphs — text-based report only
  • Mobile responsive — desktop-first
